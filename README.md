# Chordly PWA

Personal PWA for browsing song lyrics and chords, installable on iPhone and usable offline.

## Stack

- **Vite 6** + **React 18** + **TypeScript** (strict mode)
- **vite-plugin-pwa 0.21** — service worker, manifest, offline via Workbox precache
- **chordsheetjs v14** — ChordPro parsing and transposition
- **react-router-dom v6** — routing `/` → `/song/:id` → `/playlist/:id` → `/settings`
- **@dnd-kit/core + @dnd-kit/sortable** — drag & drop for playlist reordering

## Commands

```bash
npm run dev      # local development
npm run build    # production build (tsc + vite build)
npx serve dist   # test PWA/service worker locally
node generate-icons.mjs  # regenerate PNG icons in public/icons/
```

## Adding a Song

1. Create `src/songs/slug-song.json`:

```json
{
  "id": "slug-song",
  "title": "Title",
  "artist": "Artist",
  "key": "Am",
  "tags": ["italiana"],
  "content": "{title: Title}\n{artist: Artist}\n{key: Am}\n\n[Am]Song [F]lyrics..."
}
```

2. Add the import in `src/songs/index.ts` and insert it into the array.
3. `npm run build`.

The `tags` field is optional but recommended. A song can have multiple tags: `["italiana", "cristiana"]`.
Tags appear automatically in the side drawer. Default labels in `TagDrawer.tsx` (`TAG_LABELS` constant):
`"italiana"` → "Musica italiana", `"straniera"` → "Musica straniera", `"cristiana"` → "Musica cristiana".

The `content` field uses **ChordPro** format: chords in `[` `]` inline before the word.

## Playlists

- A special **Favorites** playlist (`__fav__`) is created automatically.
- Custom playlists can be created from the side drawer.
- Playlists have their own page (`/playlist/:id`) with drag & drop reordering.
- Order is persisted in `localStorage` via `usePlaylists`.

## Theme

- Toggle (☀/☾) available in SongList, SongView, and Settings.
- `useTheme` reads from `localStorage` on mount and writes `data-theme` on `<html>`.
- CSS: `:root` = dark defaults, `[data-theme="light"]` overrides all vars.

| Variable        | Dark          | Light         |
|-----------------|---------------|---------------|
| `--bg`          | `#0f0f1a`     | `#f5f5fa`     |
| `--surface`     | `#1a1a2e`     | `#ffffff`     |
| `--surface-2`   | `#252540`     | `#e8e8f5`     |
| `--text`        | `#e8e8f0`     | `#1a1a2e`     |
| `--text-muted`  | `#8888aa`     | `#555577`     |
| `--chord`       | `#f4c55a`     | `#b8680a`     |

## Notes

- Songs are bundled in the JS → offline guaranteed with no additional logic.
- The main chunk is large (~1.4 MB) because chordsheetjs includes jsPDF. Acceptable for personal use.
- PNG icons in `public/icons/` are generated by `generate-icons.mjs` (pure Node.js, no external deps).
